1. What is a Computer?
Computer is a machine that processes the input we give in the form of data and gives us the output based on the data.


2. What is RAM?
Ram also known as Random Access Memory is a storage that is used by the computer for faster processing of the data to provide us the output. RAM is the primary memory source of our computer.


3. Where is data stored in a computer?
Data is temporarily stored in the RAM and permanently stored in the HardDisk.


4. What is that input device used to type text and numbers on a document in the computer system?
Keyboard.


5. What are the output devices?
The devices that give us the output in a graphic or physical medium are known as output devices . For e.g: Monitor, Printer.


6. Which is the input device that allows a user to move the cursor or pointer on the screen? 
The mouse.


7. Which language is directly understood by the computer without a translation program?
Machine Level Language which is 0’s and 1’s.


8. What are Input devices?
Devices which we use to provide inputs to our computer are known as Input devices.
